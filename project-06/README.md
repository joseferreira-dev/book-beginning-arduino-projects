# Interactive LED Chase Effect

The objective of this project is to create a sequential lighting effect using LEDs and interacting with it through a potentiometer.

## Project Components

- 1x Arduino UNO
- 1x Protoboard
- 10x 5mm LED
- 10x 150Ω resistor
- 1x 4.7kΩ potentiometer
